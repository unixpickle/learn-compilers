# Sudoku solver

Build the executable like

```
swift run Compile code out.s
clang out.s
```

And run an example board like

```
cat example_board.txt | ./a.out
```

In that case, the output looks like

```
254316897
763985124
198427653
981753246
632849715
547261938
475692381
319578462
826134579
```
